# Linguagem de Programação - Python 🐍

Repositório dedicado ao estudo da linguagem Python, cobrindo desde sintaxe básica até estruturas avançadas de programação funcional e modularização.

## 📑 Conteúdo
- **Unidade 1:** Introdução, variáveis e tipos de dados.
- **Unidade 2:** Estruturas de decisão e repetição.
- **Unidade 3:** Funções (`def`), Parâmetros (*args e **kwargs) e Expressões Lambda.
- **Unidade 4:** Manipulação de arquivos e bibliotecas.

## 🛠️ Tecnologias
- Python 3
- Google Colab
